/lit/ Leaderboard

This leaderboard uses several exhaustive methods to scrape amazon paperback links and sort them by Best Sellers Rank.

Amazon employs very strict anti-bot measures. This tool could break at any moment. This tool is slow, a feature, not a bug.

If you want to add your book to the leaderboard, please use the submission form below. PLEASE ONLY ENTER VALID AMAZON PAPERBACK URLs.

Please submit one book per day. Please only submit books that you yourself have written or published. Please respect this tool, it is fragile.

To update the leaderboard, press the Cycle button.

A stable version of this tool can be downloaded from GitHub.
